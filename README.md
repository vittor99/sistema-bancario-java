# Sistema Bancário Java

Projeto desenvolvido em Java puro com objetivo de praticar:

- Programação Orientada a Objetos
- Encapsulamento
- Herança
- Polimorfismo
- Collections
- Exceções
- Organização em camadas

## Tecnologias

- Java
- IntelliJ IDEA

## Objetivo

Criar um sistema bancário utilizando boas práticas de desenvolvimento e evoluir posteriormente para uma aplicação com Spring Boot e banco de dados.
